# Pixi.js boilerplate
  - used as a starting point
  
## Features
- ES6 (ES2015) support
- Gulp build system
- Browserify with watchify
- Notifications on build errors
- Watch task with local webserver
- Require support for shaders (.vert, .frag, .glsl)
